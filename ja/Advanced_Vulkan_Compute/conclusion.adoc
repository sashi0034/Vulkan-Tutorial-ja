= Advanced Vulkan Compute: まとめ

「Advanced Vulkan Compute」チュートリアルシリーズの完走、おめでとうございます。コンピュートシェーダの基本概念から、Vulkan 1.4 における高性能 GPGPU 開発の最前線までたどり着きました。

== 学んだこと

本シリーズでは、単純な画像処理を超えて、複雑でヘテロジニアスなシステムへと進み、現代 GPU コンピュートの深層を探りました。

1. **コンピュートアーキテクチャ**: ワークグループグリッドと物理ハードウェア（CU と SM）のマッピングを習得し、占有率の最大化とレイテンシ隠蔽を学びました。また、帯域効率最大化のために Vulkan 1.4 のスカラーレイアウトも活用しました。
2. **メモリモデル**: Vulkan Memory Model を解明し、可用性・可視性・ドメイン操作を習得して、性能を犠牲にせずスレッド安全を確保しました。
3. **サブグループの力**: サブグループシャッフル、ブロードキャスト、算術演算を使い、VRAM や共有メモリ（LDS）を完全に迂回してハードウェア速度でデータを交換しました。
4. **ヘテロジニアスエコシステム**: `clspv` と `clvk` でレガシー OpenCL コードを Vulkan へ橋渡しする手法を探りました。`clvk` は Vulkan 上にレイヤされた適合 OpenCL 3.0 を提供し、アプリケーションからは見えません。
5. **高度なデータ構造**: GPU 常駐の木、ロックフリーリンクリストを実装し、ポインタのような柔軟性のために生の Buffer Device Address（BDA）を活用しました。
6. **GPU 駆動パイプライン**: インダイレクトディスパッチと自律的なコマンド生成により、制御を CPU から GPU へ移しました。
7. **非同期オーケストレーション**: Synchronization 2 とタイムラインセマフォを使い、複数のハードウェアエンジンの力でコンピュートをグラフィックスと並行実行しました。
8. **専用数学**: コオペラティブ行列や混合精度（FP16/Int8）といった現代ハードウェア機能を使い、大規模なスループットを得ました。
9. **パフォーマンスと診断**: 発散についてカーネルを監査し、Roofline モデルでスループットを分析し、GPU-Assisted Validation（GAV）で複雑な競合状態をデバッグする方法を学びました。
10. **AI 支援最適化**: 大規模言語モデル（LLM）が、素朴な逐次ロジックと、高い GPU スループットに必要な並列・サブグループ意識のパターンとの橋渡しになり得ることを見ました。
11. **モバイル＆組み込みコンピュート**: これらの技法を、最も広く展開されている Vulkan ハードウェア——Android スマートフォン、シングルボードコンピュータ、自動車、ロボティクス——へ持ち込み、電力と帯域を最適化し、ユニファイドメモリを活用し、ディスプレイのないデバイスでヘッドレス実行しました。

== 要点を押さえる: コンピュートのマインドセット

本シリーズからひとつだけ持ち帰るなら、これです。**GPU は単なる数学コプロセッサではなく、独自の複雑な階層を持つ並列スループットマシンである。**

高度なコンピュート開発には、思考の転換が必要です。

- **ウェーブで考える**: ワークグループレベルのバリアの代わりに、サブグループ（Wave）演算を使う機会を常に探す。
- **明示的な同期**: 運に頼らない。Vulkan Memory Model と Synchronization 2 で、データがいつ・どのように可視になるかを正確に定義する。
- **データファースト設計**: ロジックを一行も書く前に、GPU のメモリアーキテクチャ（スカラーレイアウト、LDS バンク整列）に合わせてデータ構造を設計する。

== 次のステップ

高性能コンピューティングの世界は広大です。確かな基盤ができた今、次の道を検討してみてください。

1. **機械学習への深掘り**: コオペラティブ行列と混合精度で学んだことを使い、ニューラルネットワークの推論や学習を最適化する。
2. **リアルタイムパストレーシング**: GPU 駆動パイプラインと非同期コンピュートを組み合わせ、複雑な空間構造をデバイス上だけで扱う高性能レイトレーサを構築する。
3. **物理シミュレーション**: 議論したロックフリーデータ構造を使い、高度な流体力学（SPH）や剛体ソルバを実装する。
4. **Vulkan エコシステム**: `clspv` や `clvk` のようなプロジェクトに貢献するか、本シリーズで扱った Vulkan 機能の上に独自の高レベルコンピュート抽象を構築する。

この旅に一人ではありません。Vulkan コミュニティには専門家と熱意ある人が揃っています。

- **Vulkan 仕様**: メモリモデルとハードウェア制約に関する最終的な情報源。
- **ベンダー固有ドキュメント**: デスクトップ向けのハードウェア固有最適化には、NVIDIA の Nsight、AMD の RGP、Intel の GPA ドキュメントを参照してください。モバイルと組み込み向けには、Arm の Mali/Immortalis ベストプラクティスガイドと Performance Studio、Qualcomm の Adreno 開発者ドキュメントと Snapdragon Profiler、Imagination の PowerVR 開発者リソースを参照してください。

本シリーズに付き合ってくださりありがとうございます。「ピクセルをきれいにする」段階から、現代ハードウェアの並列パワーをフルに活用する段階へ進みました。あとは想像力だけが限界です。
