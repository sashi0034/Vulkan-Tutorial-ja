= ライティングとマテリアル: 基本的なライティングモデルとプッシュ定数

この章では、基本的なライティングモデルの実装と、Vulkan でマテリアルプロパティにプッシュ定数を使う方法を扱います。エンジン実装全体を通じて、モダンで効率的かつ保守可能なコードベースを作るため、vk::raii ダイナミックレンダリングと C++20 モジュールを使います。
