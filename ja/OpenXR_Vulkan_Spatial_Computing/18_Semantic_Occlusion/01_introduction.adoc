= セマンティックオクルージョン: 深度認識の現実／仮想合成

これまでの章では、物理世界の幾何を取り込む方法を学びました。しかし説得力のある体験には、その幾何を持つだけでは不十分です。仮想ロボットを床に置くと、現実のコーヒーテーブルがあなたとロボットの間にあるとき、隠れるべきです。これが **セマンティックオクルージョン（Semantic Occlusion）** の課題です。

この章では、深度認識オクルージョンと **セマンティックセグメンテーション** を通じて、現実と仮想の世界のギャップをどう埋めるかを探ります。ML モデルで現実をピクセル単位にラベル付けし——「人間の手」と「テーブル」を区別し——Vulkan の **ステンシルバッファ** と **深度認識合成** を使い、仮想物体が現実の物体の「後ろに収まる」ことをサブピクセル精度で保証する方法を学びます。

この章の終わりまでに、仮想アセットがユーザーの手や家具と同じ物理空間を真に占めるように感じさせる、安定でフリッカのないマスキングを実装できるようになるでしょう。
