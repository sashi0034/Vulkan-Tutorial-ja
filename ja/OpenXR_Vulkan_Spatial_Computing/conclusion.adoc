= おわりに

**OpenXR と Vulkan 1.3 による空間コンピューティング** チュートリアルシリーズの完走、おめでとうございます！ 2 つの API 間の最初のハンドシェイクから、LightField Theory や Plenoptic Synthesis といった発展的な話題まで、広大な範囲をカバーしてきました。

各章末の「エンジンへの組み込み」セクションを通じて、これらの理論的概念を `simple_game_engine` のような実世界のレンダラーにどう適用するかを見てきました。

空間コンピューティングは絶えず進化する分野です。Vulkan 1.3 と OpenXR で可能なことの限界を、ぜひ探索し、実験し、押し広げ続けてください。
