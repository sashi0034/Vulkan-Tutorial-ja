= シーングラフとトランスフォーム階層

== 世界の統一

「シンプルなエンジンの構築」シリーズでは、ハイブリッドなシーンアーキテクチャを実装しました。ワールド内に `ObjectInstance` エンティティのフラットなコレクションを維持し、それぞれが自身の内部 glTF ノード階層を持つ `Model` を含んでいました。これは学習には完璧な中間地点で、複雑な glTF アセットを読み込みつつ、トップレベルのエンジンロジックを単純に保てました。

しかし高度なキャラクターアニメーションと物理へ進むと、この分離は障壁になります。キャラクターは自己完結したモデルではなく、相互作用するシステムの集合です。手が別の剣エンティティを掴む必要があるノードかもしれないし、足が地形の物理コライダーと相互作用しなければならないノードかもしれません。こうしたエンティティ横断の相互作用をシームレスに支えるには、**統一シーングラフ** へ移行する必要があります。

本章では、「世界」と「モデル」の境界を溶かします。200 ボーンのキャラクターでも静的な建物でも光源でも、すべてが普遍的階層内の単なる `Node` である、単一のグローバルな再帰ノードシステムへエンジンを移行します。

== 性能の障壁: 冗長な計算

「シンプルなエンジン」は、ワールド空間の変換行列を計算するために素直な再帰トラバーサルを使っていました。

[source,cpp]
----
void renderNode(const std::vector<Node*>& nodes, const glm::mat4& parentMatrix) {
    for (const auto node : nodes) {
        glm::mat4 nodeMatrix = parentMatrix * node->getLocalMatrix();
        // ... render and recurse ...
    }
}
----

優雅ではありますが、このアプローチには重大な性能上の欠陥があります。何も動いていなくても、毎フレームすべてのノードについてワールド行列を再計算することです。数千ノードのシーン—モダンなキャラクター多用環境では珍しくありません—では、CPU サイクルとメモリ帯域の大幅な浪費になります。

これを解くために **Dirty Flag パターン** を実装します。各ノードは、ローカルトランスフォームまたは親のワールドトランスフォームが変わったかどうかを追跡します。ノードが「dirty」なときだけ、高価な行列乗算を行います。この最適化は、数キャラクターで詰まるエンジンと、群衆全体を扱えるエンジンの差です。

== アニメーションと物理の橋渡し

シーングラフは、キャラクターパイプラインのアーキテクチャ上の「神経系」です。スケルタルアニメーションの数学理論と、エンジン世界の物理的現実が出会う場所です。

キャラクターのスケルトンは本質的に、特殊化されたシーングラフにすぎません。まず堅牢で最適化された階層を構築することで、単純な親子アタッチメントも複雑なスケルタル構造も、同じ基盤ロジックで扱えるシステムができます。さらにこの階層は物理エンジンへのフックを提供します。衝突プロキシ（カプセル、ボックス）を階層内のノードに直接アタッチします。アニメーションシステムがボーンノードを動かすと、物理エンジンのコライダーが自動で追従します。この双方向同期により、スイングの最中でもキャラクターの腕が衝突に反応できます。

== 何を構築するか

本章では、高度なキャラクターパイプラインの基盤「配管」を、四つの主要領域にわたって敷設します。

*   **エンジン拡張**: エンティティ管理をグローバルな再帰 `Node` システムへリファクタします。Dirty Flag 最適化を実装し、キャッシュ局所性を改善するよりデータ指向のレイアウトを探ります。
*   **メタデータと物理 extras**: glTF の "extras" を使い物理セットアップを自動化する方法を探ります。3D ツールで定義したカスタムプロパティを活用し、エンジン内でコライダーと物理プロパティを自動生成します。
*   **物理同期の理論**: 視覚ノードと物理リジッドボディの数学的リンクを確立し、シリーズ後半の Kinematic-to-Dynamic ハンドオフ（ラグドール）に備えます。
*   **変換パイプライン**: 逆変換とグローバルからローカルへの変換を扱うよう数学を洗練します。これはインバースキネマティクス（IK）の前提条件です。

まずエンジンのコアを拡張し、モダンなキャラクターアニメーションの厳しさに耐えるシーングラフを構築するところから始めましょう。
